# orvox

Чтобы запустить orvox:

```bash
cp .env.example .env
docker compose up -d
```

минимальные системные требования:
  * Docker
  * Docker Compose
  * 4 ГБ RAM
  * 16 ГБ Дискового пространства

Рекомендуемые системные требования:
  * Docker
  * Docker Compose
  * Postgres 17
  * Coturn
  * 8 ГБ RAM
  * 16 ГБ Дискового пространства
